Adding Visual YoY Changes Chart 25

Steps

  • First create a measure description table.
  • On the Power BI Desktop Home ribbon, Click on Enter data, name the first column header as ID and the second column header as Measure and Name the table as Google Play Store Impact. In the table, insert ID(1,2,3,4,5) and Measure(YoY Installs, YoY Revenue, YoY Apps, YoY Rating) respectively as shown below, then Click ok, Close & apply. 

  • Create New Measure 
YoY Changes = SWITCH(
  SELECTEDVALUE('Google Play Store Impact'[ID]),
    1, [Total Installs YoY],
    2, [Total Revenue YoY],
    3, [Total Apps YoY],
    4,[Avg Rating YoY])
Follow the Gif below



  • Click on an empty space on the canvas, and from the visualizations pane select Waterfall Chart
  •  Fill up figures provided below for the necessary fields.
  • Build Visual Tab:
  • Category= Measure (Google Play Store Impact) , Column y-axis= YoY Changes (Google Play Store Impact)
  • Tooltips = Total Install YoY, Total Revenue YoY, Total Apps YoY, Avg Rating YoY.
Format Visual Tab:
Visual:
  • X-axis: Values: Font=Segoe UI (9), Font Color=#605E5C, Title = On     
  • Y-axis: Values: Font=Segoe UI Bold (9), Font color=#605E5C, Display Units=Auto, Title = On           
  • Legend=On, Position= Top left, Text= Segoe UI (10), Color=#605E5C,Title=off
  • Gridlines: Horizontal: Off.
  • Column Color = Increase= #1AAB40, Decrease= #D64554, Show total column = On, 
  • Data labels: On, Horizontal. Values = Segoe UI (9), Bold, Color = #000000(Black), Display Unit= Auto, Value Decimal=0  Background= Off.
General:
  • Properties: Size, Height =320, Width =534, Position, Horizontal = 743, Vertical =400,
  • Title= On, Text= YoY Changes, Font=Din (14), Text color=#118DFF, Background color=No Fill, Horizontal alignment=Center
  • Effects: Background=On, Background color=#FFFFFF, Transparency=25%,
  • Visual boarder=Off, Shadow =Off
  • Tooltips=On, Font= Segoe UI (9), Color= #C8C8C8, Value color=#EAEAEA, Background color= #333333, Transparency= 5%
